# FitNexus - 헬스장 PT 회원 관리 시스템

헬스장 PT 회원 관리 및 PT 선생 평가 시스템

## 기능

- 📋 PT 회원 관리 (등록, 조회, 수정)
- ⭐ PT 선생 평가 시스템
- 📊 평가 통계 및 분석
- 🎨 모던한 UI/UX

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.
